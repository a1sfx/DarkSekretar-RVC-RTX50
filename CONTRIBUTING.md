# Contributing to DarkSekretar-RVC-RTX50

## Как помочь проекту

Если вы нашли баг или у вас есть предложение по улучшению:

1. **Создайте Issue** — опишите проблему или идею
2. **Форкните репозиторий** и создайте ветку с вашими изменениями
3. **Сделайте Pull Request** с описанием того, что вы изменили

## Основные направления для помощи

- Тестирование на других картах RTX 50 series (5070/5080/5090)
- Адаптация для AMD/Intel GPU
- Исправление оставшихся багов WebUI
- Оптимизация использования памяти
